I build web interfaces with TypeScript, React, and Tailwind CSS, from individual components to full systems. My work covers the entire frontend layer: component architecture, state management, REST API integration, and responsive UI delivery. On the backend, I develop with PHP and Laravel, with relational modeling in PostgreSQL, user authentication, access control, and RESTful APIs consumed by the frontend itself.

Frontend Engineer at [Asymptora](https://asymptora.com), where I own the organization's entire web platform. The main project is asymptora.com: a multi-author system built on Ghost CMS with a fully custom theme in React and Tailwind CSS, covering authentication and session management, individual user spaces with role-based access control, a publishing platform where each member manages their own blog, and product interfaces integrated with the backend team's Java REST APIs. Those interfaces include a healthcare application going into production on a Kubernetes cluster operated by the Platform Engineering team.

I care about the full path from code to user. When I build a component, I think about the API contract behind it. When I design an interface, I think about the data model it depends on. I want to understand what happens after the deploy, not just before it.
